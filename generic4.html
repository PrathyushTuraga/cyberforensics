Module 4


↠1.1 Description

In this incident, we will install the modified APK file on mobile device (or) a AVD (Android Virtual Device) on Android Studio. Also, we will setup web server with Python on local system.


↠1.4 Instructions

  • Step1: Email the login.apk file provided under "Incident 3: Artifacts" to the target device.
  • Step2: On the device, download attachment from email, click install and then open the app.
  • Step3: If you are using Android Studio, create a new Android Virtual Device (AVD).

    •    https://www.youtube.com/watch?v=x_lvdLil0Fk

       
       (Note2: Recommended configurations for device are Pixel 3 API 27 (Oreo) - Android 8.1. You can see these configurations while setting up AVD)
  • Step4: When device is ready, you can see android home screen as shown in below screenshot


  • Step5: Go to apps page on device inside emulator window. Now drag & drop login.apk file from your local file system onto the device inside emulator.

    •    https://www.youtube.com/watch?v=UtkYGh7EE3A

       (Note: You can refer this link for steps to install app on AVD in android studio)
  • Step6: After installation, open app and observe the pages

  • Step7: Next step is to setup server on your local computer. Make sure python is installed on your computer for setting up web server.

  • Step8: Open new command prompt and execute below command according to python version installed. Use port 8080 for the http server.

    •    (For Python 2.X -> python -m SimpleHTTPServer 8080)
       (For Python 3.X -> python -m http.server 8080)
